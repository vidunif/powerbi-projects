# Amazon Sales Analysis Dashboard | Power BI

## Project Overview
This project is an interactive Power BI dashboard built to analyze Amazon product sales and performance using key business metrics and visual insights.

## Tools & Technologies
- Power BI
- Power Query
- DAX (Basic)
- Excel

## Key Insights
- Year-to-Date (YTD) and Quarter-to-Date (QTD) sales performance
- Monthly and weekly sales trends
- Product category-wise sales contribution
- Top 5 products by sales and customer reviews
- Interactive filters for category and quarter analysis
